\subsection{Completando Números}

\paragraph{} Ao trabalhar com os tempos , muitas vezes você vai encontrar valores float com muitos dígitos após o decimal. Para tornar estes valores mais fácil trabalhar, você pode abreviá-los com a função de Python built-in round( ), que arredonda um flutuador com a precisão que você especificar. Basta passar o número que você deseja arredondar mais um segundo argumento opcional que representa quantos dígitos depois da vírgula você deseja arredondar para ele. Se você omitir o segundo argumento, a round() arredonda o número para o inteiro mais próximo, veja o exemplo:

\subsection{Projetando um Super Cronômetro}

\paragraph{} Suponhamos que você queira automatizar tarefas relacionadas a tempo, realmente chatas. Assim, suponhamos também que não encontre um aplicativo para alguma das interfaces que usa no dia-a-dia. Então, como criar um super cronômetro para fazer algumas tarefas para você?
	
\vspace{0.2cm}
Podemos assim, seguir os passos. Em alto nível, seu programa deverá:
	
\vspace{0.2cm}
Imprimir o número da volta, tempo total e tempo de volta.
E isso significa que seu código terá que fazer também:
		 
\vspace{0.2cm}
\begin{enumerate}
\item Encontrar o tempo atual chamando time.time( ) e armazená-lo como um timestamp no início do programa , bem como ao início de cada volta.
		
\vspace{0.2cm}
\item Mantenha um contador de voltas e incrementá-lo cada vez que o usuário pressiona enter.
		
\vspace{0.2cm}
\item Calcule o tempo decorrido subtraindo timestamps.
		
\vspace{0.2cm}
\item Tratar a exceção KeyboardInterrupt de modo que o usuário pode pressionar ctrl- C para sair, abrir uma nova janela do editor de arquivo e salvá-lo como stopwatch.py.
		
\vspace{0.2cm}
Ok, tudo isso parece difícil correto? Não, vamos a seguir estudar cada parte que necessitamos saber para resolver nosso problema.

\paragraph{} Para introduzir o conceito de multithreading , vejamos um exemplo. Digamos que você queira agendar algum código para executar depois de um atraso algum tempo, você pode adicionar código como o seguinte no início do seu programa:

\paragraph{} Este código designa uma hora de início de 31 de Outubro de 2029, e continua chamando time.sleep (1) até que chega a hora de início. Seu programa não pode fazer nada enquanto espera para o loop de time.sleep( ). Ao invés de ter todo o seu código esprando até que a função time.sleep( )
termine você pode executar o código separadamente usando o módulo de threading do Python. O segmento separado fará uma pausa para as chamadas time.sleep. Enquanto isso, seu programa pode fazer outro trabalho no thread original. Dessa forma, você leitor precisará inserir o seguinte código de um novo arquivo e salvá-lo como threadDemo.py, assim:
		
\vspace{0.25cm}
\includegraphics[width=7cm]{figura10.png}

\paragraph{} Não podemos nos confundir, se print ('Fim do programa') é a última linha do programa, você pode pensar que ele deve ser a última coisa impressa. Não, pois Wake up! vem depois dela é que quando threadObj.start( ) é chamado. O alvo de threadObj é executado em um novo segmento de execução. Pense nisso como um segundo dedo que aparece no início da função takeANap ( ). O principal segmento continua a imprimir ('fim do programa'). Enquanto isso, o novo segmento que foi executar o time.sleep (5) faz uma pausa durante 5 segundos. Depois disso, acorda de sua soneca de 5 segundos e ele imprime 'Wake up!' e, em seguida, retorna a partir da função takeANap( ). Cronologicamente, 'Acorde!' é a última coisa impressa pela
o programa.
\paragraph{} Normalmente, um programa termina quando a última linha de código no arquivo foi executado(ou a função sys.exit( ) é chamada). Mas threadDemo.py tem dois tópicos. O primeiro é o segmento original que começou no inicio do programa e termina após print('Fim do programa') e o segundo segmento é criado quando threadObj.start( ) é chamado, começa no início da função takeANap( ), e termina após takeANap( ). Um programa Python não terminará até que todos os seus segmentos tenham terminado. Quando você executou threadDemo.py, embora o segmento original tinha rescindido, o segundo segmento ainda estava executando o time.sleep(5).

\section{LANÇAMENTO DE OUTROS PROGRAMAS NO PHYTON}
		
Seu programa Python pode começar outros programas no seu computador com a função Popen( ). Se você tiver várias instâncias de uma aplicação aberta, cada um desses exemplos é um processo separado da mesmo programa. Por exemplo, se você abrir várias janelas do seu navegador ao mesmo tempo, cada uma dessas janelas é um processo diferente do browser.
